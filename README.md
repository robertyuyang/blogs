# 移动开发技术积累 
> 该仓库用于巩固移动开发组的技术积累。
> 
> 不限于技术文章和技术分享文档，**任何技术相关的东西都可以往这个仓库里放**，供组内同学学习和自己日后复习。
> 
> 包括但不限于：
> 
> + 技术分享文档、自己编写的、翻译的代码规范、技术文章。
> 
> + 跟业务相关的wiki、文档，不适合放到具体项目的仓库里的，也可以放到这个仓库里
>
> + 一些能作为技术积累的demo代码。
> 
> + 读到的一些别人写的有技术含量的文章、技术文档、甚至觉得有价值的一整个blog首页或者GitHub仓库地址，都可以链接进来。
>
